Introduction about the game


Releashed in 2012

Counter-Strike: Global Offensive is a tactical multiplayer first-person shooter computer game developed by Valve Corporation and Hidden Path Entertainment, which has also continued to maintain Counter-Strike: Source after release. This is the fourth game in the popular Counter-Strike series. If not counting the two online versions Counter-Strike Online (2007) and Counter-Strike Online 2 (2013).

The game is developed based on the original content from Counter-Strike 1.6. Such as revamped versions of classic maps, weapons. As well as adding new maps, characters and game modes. Multi-platform multiplayer game was planned between Windows, OS X and PSN players, but was ultimately limited to Windows and OS X because of the difference in the number of updates between the systems.

How to play


Step 1

Sign in with your personal Steam account


Step 2 - Go to Store

Search Counter-Strike: Global Offensive in the search bar


Step 3

Chose CS:GO, and press "Play game"


Step 4

Select the drive to install. If you don't have a Steam Library folder, you need to create a new folder (default folder is C drive). Choose the path to save game data through the option "Choose location for install"


Step 5: Wait for Steam to complete the scan and click Finish

After the game has been download successfully, open the game and let's play CS: GO
